# Pirate Treasure Quest

Pirate Treasure Quest is a beginner-friendly Python adventure game inspired by Karel-style logic puzzles. The player controls a pirate exploring a mysterious 7x7 island filled with hidden traps, coins, extra lives, and a secret Golden Key needed to unlock the treasure.

The game is built to help beginners practice programming concepts such as loops, lists, conditionals, and functions while having a fun interactive experience.

---

## Game Description

You start your journey as a pirate stranded on an unknown island. Legends say a powerful treasure lies hidden at the far corner of the map. However, the island is dangerous and filled with traps that can send you back to the start.

To succeed, you must explore carefully, collect coins, find extra lives, and locate the Golden Key. Only then can you unlock the treasure and win the game.

Each playthrough generates a new map, making every game different.

---

## Features

- 7x7 grid-based map system
- Fog-of-war exploration (unvisited tiles are hidden)
- Keyboard movement using W, A, S, D
- Randomly generated traps, coins, and hearts
- Golden Key required to unlock treasure
- Extra lives system
- Coin collection system
- Shop system (buy lives with coins)
- Reset mechanic when hitting traps
- Win and lose conditions
- Fully replayable random maps
